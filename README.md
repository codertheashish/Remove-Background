# 🖼 Background Remover

A simple and fast Python-based Background Remover that removes the background from any image using AI. Just provide an image path, and it automatically generates a clean output image without background.

---

## 🚀 Features

* 📥 Input image directly from local system
* 🧠 AI-based background removal
* 🖼 Supports common image formats (JPG, PNG, etc.)
* 💾 Saves output as `output_ashish.png`
* ⚡ Lightweight and beginner-friendly

---

## 🛠 Tech Stack

* **Python 3.x**
* **withoutbg** library

---

## 📦 Installation

1️⃣ Install the required module:

```bash
pip install withoutbg
```

2️⃣ Clone :

```bash
https://github.com/codertheashish/Remove-Background
```

---

## ▶️ How to Use

1. Run the script:

```bash
python remove_background.py
```

2. Make sure your image (e.g., `ashish.jpeg`) is in the same folder

3. The script will process the image automatically

4. Output will be saved as:

```
output_ashish.png
```

---

## 📁 Output

* ✔ Background removed image
* ✔ Saved locally as `output_ashish.png`

---

## 📌 Notes

* Input image path should be correct
* Works best with clear subject images
* Supports Windows, macOS, and Linux
